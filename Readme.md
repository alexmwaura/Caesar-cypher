# scrabble score
A simple java application for the Caesar cypher that encrypts and decrypts any given word or sentence

## Prerequisites
    1. java 11
    2. Gradle 4.10

## Usage

    1. Clone this repo
    2. open with intellij IDEA
    3. Then run ` javac App.java && java App`


## Testing

   ```java
    gradle test
```
